# Data

Raw data comes from the official City of Houston 311 Service Request Data page:

https://www.houstontx.gov/311/servicerequestdata.html

This project uses:

- `2025 (piped)`
- `2026 YTD (piped)`

The raw extracts are large and are not committed to GitHub. Download them into `data/raw/` before running the analysis:

```text
data/raw/houston_311_2025_piped.txt
data/raw/houston_311_2026_ytd_piped.txt
```

Processed summary files generated by the analysis are stored in `data/processed/`.
